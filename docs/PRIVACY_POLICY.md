# Vita AI 隐私政策

**生效日期**: 2024年1月1日  
**最后更新**: 2024年1月1日

## 简介

Vita AI ("我们"、"我们的") 重视您的隐私。本隐私政策说明我们如何收集、使用、披露和保护您的个人信息。

本政策符合新加坡《个人数据保护法》(PDPA) 的要求。

## 我们收集的信息

### 1. 您提供的信息

**健康画像信息**:
- 年龄
- 性别
- 身高和体重
- 活动水平
- 健康目标

**联系信息**:
- WhatsApp 电话号码
- 语言偏好

**使用数据**:
- 食物照片
- 食物记录
- 营养数据
- 使用统计

### 2. 自动收集的信息

**技术信息**:
- 设备类型
- 操作系统
- IP 地址
- 使用时间和频率

**交互数据**:
- 命令使用记录
- 功能使用情况
- 错误日志

## 我们如何使用您的信息

### 主要用途

1. **提供服务**
   - 食物识别和营养分析
   - 生成个性化建议
   - 发送每日健康总结

2. **改进服务**
   - 提高识别准确性
   - 优化用户体验
   - 开发新功能

3. **账号管理**
   - 验证身份
   - 处理订阅
   - 提供客户支持

4. **安全保护**
   - 检测异常活动
   - 防止欺诈
   - 保护账号安全

### 次要用途

1. **分析和研究**
   - 匿名统计分析
   - 服务性能监控
   - 用户行为研究

2. **营销通信**（需您同意）
   - 产品更新通知
   - 促销活动信息
   - 健康小贴士

## 信息共享

### 我们不会出售您的数据

我们**绝不会**出售、出租或交易您的个人信息。

### 有限的信息共享

我们仅在以下情况下共享信息：

1. **服务提供商**
   - OpenAI (食物识别)
   - Stripe (支付处理)
   - Supabase (数据存储)
   - Vercel (应用托管)
   - Upstash (缓存服务)

   这些提供商受合同约束，只能按我们的指示使用数据。

2. **法律要求**
   - 遵守法律义务
   - 响应法院命令
   - 保护权利和安全

3. **业务转让**
   - 合并或收购时（需通知您）

## 数据安全

### 安全措施

我们采用多重安全措施保护您的数据：

**技术措施**:
- 🔒 AES-256 加密存储
- 🛡️ HTTPS 传输加密
- 🔐 访问控制和身份验证
- 🚨 异常活动监控

**组织措施**:
- 👥 员工培训
- 📋 安全政策和程序
- 🔍 定期安全审计
- 📝 事件响应计划

### 照片处理

**照片安全流程**:
1. 上传到安全服务器
2. AI 分析（5-10 秒）
3. 生成营养数据
4. **自动删除**（24 小时内）

我们**不会长期保存**您的食物照片。

## 数据保留

### 保留期限

| 数据类型 | 保留期限 | 原因 |
|---------|---------|------|
| 健康画像 | 账号存续期间 | 提供服务 |
| 食物记录 | 账号存续期间 | 历史查询 |
| 照片 | 24 小时 | 临时处理 |
| 使用日志 | 90 天 | 故障排查 |
| 支付记录 | 7 年 | 法律要求 |

### 删除数据

您可以随时删除数据：

```
/delete history   # 删除历史记录
/delete account   # 删除账号和所有数据
```

**注意**: 删除后无法恢复！

## 您的权利

根据 PDPA，您有以下权利：

### 1. 访问权

查看我们持有的您的个人数据：
```
/data export      # 导出所有数据
```

### 2. 更正权

更新不准确的信息：
```
/profile          # 更新健康画像
```

### 3. 删除权

删除您的个人数据：
```
/delete account   # 删除所有数据
```

### 4. 限制处理权

限制某些数据处理：
```
/settings         # 管理数据使用偏好
```

### 5. 数据可携权

以结构化格式导出数据：
```
/export csv       # CSV 格式
/export pdf       # PDF 格式 (Pro)
```

### 6. 反对权

反对某些数据处理：
```
/settings         # 关闭营销通信
```

### 7. 撤回同意权

随时撤回同意：
```
/consent          # 管理同意设置
```

## 儿童隐私

Vita AI 适合 13 岁以上用户。

**13 岁以下儿童**:
- 需家长同意
- 家长监督使用
- 家长可访问和删除数据

如果我们发现未经同意收集了儿童数据，将立即删除。

**数据中心位置**:
- 🇸🇬 新加坡（主要）
- 🇺🇸 美国（备份）

我们确保所有数据传输符合 PDPA 要求，并采用适当的保护措施。

## 第三方链接

Vita AI 可能包含第三方网站链接。我们不对这些网站的隐私实践负责。请查看他们的隐私政策。

## 数据泄露通知

如果发生数据泄露：

1. **评估**: 立即评估影响
2. **通知**: 72 小时内通知受影响用户
3. **补救**: 采取措施防止进一步泄露
4. **报告**: 向 PDPC 报告（如需要）
