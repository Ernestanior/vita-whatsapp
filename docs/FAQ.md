## 入门问题

### 如何开始使用 Vita AI？

1. 保存 Vita AI 的 WhatsApp 号码
2. 发送 "Hi" 或 "/start"
3. 按照提示设置您的健康画像
4. 开始发送食物照片！

### 需要下载 App 吗？

不需要！Vita AI 完全通过 WhatsApp 运行，无需下载额外的应用程序。

### 支持哪些语言？

目前支持：
- 🇬🇧 English
- 🇨🇳 简体中文
- 🇹🇼 繁体中文

发送 `/language` 切换语言。

### 是否免费？

是的！我们提供免费计划，每月 10 次食物识别。如需更多功能，可升级到 Premium ($9.90/月) 或 Pro ($29.90/月)。

### 适合哪些人使用？

Vita AI 适合：
- 🏃 想要减重或增重的人
- 💪 健身爱好者
- 🩺 需要控制饮食的人（糖尿病、高血压等）
- 🥗 想要养成健康饮食习惯的人
- 👨‍👩‍👧‍👦 关注家人健康的人

## 功能问题

### 食物识别准确吗？

Vita AI 使用先进的 AI 技术，准确率约 85-90%。对于新加坡常见食物，准确率更高。如果识别不准确，您可以手动修改。

### 可以识别哪些食物？

**支持的食物类型**:
- 🍚 新加坡本地美食（鸡饭、叻沙、肉骨茶等）
- 🍜 亚洲料理（中餐、日料、韩料、泰餐等）
- 🍔 西餐和快餐
- 🥗 沙拉和健康餐
- 🍰 甜点和饮料
- 🍎 水果和零食

**暂不支持**:
- 复杂的混合菜肴（可能识别为多个食物）
- 自制的特殊菜肴
- 非常模糊的照片

### 识别需要多长时间？

通常 5-10 秒。高峰时段（午餐和晚餐时间）可能需要稍长时间。

### 一张照片可以识别多个食物吗？

可以！Vita AI 会自动识别照片中的所有食物项目。

### 如何提高识别准确性？

**拍摄技巧**:
- ✅ 确保光线充足
- ✅ 正面拍摄食物
- ✅ 食物清晰可见
- ✅ 包含参照物（如餐具、手）

**补充信息**:
发送照片后，可以补充说明：
```
"这是大份"
"这是半份"
"这是儿童餐"
```

### 健康评级是如何计算的？

健康评级基于多个因素：
- 🔢 营养成分（卡路里、蛋白质、碳水、脂肪、钠）
- 🎯 您的健康目标（减重/增重/维持）
- 📊 您的每日营养目标
- ⚖️ 营养均衡性

**评级标准**:
- 🟢 绿灯: 营养均衡，符合目标
- 🟡 黄灯: 某些营养素偏高/偏低
- 🔴 红灯: 高热量/高脂肪/高钠

### 每日总结什么时候发送？

每天晚上 9:00 (SGT) 自动发送。

**总结内容**:
- 今日营养摄入汇总
- 目标达成情况
- 健康洞察和建议
- 推荐的运动量

### 可以关闭每日总结吗？

可以。发送 `/settings` 并选择 "关闭每日总结"。

### 如何查看历史记录？

```
/history          # 查看所有记录
/history today    # 今天的记录
/history week     # 本周的记录

```

**Pro 用户**: 可以导出 CSV 和 PDF 格式

```
/export csv       # 导出 CSV
/export pdf       # 导出 PDF (Pro only)
```

### 如何更新健康画像？

**方法 1: 使用命令**
```

**方法 2: 自然语言**
```

可以！定期更新体重，Vita AI 会自动追踪变化趋势。

```

## 隐私和安全

### 我的数据安全吗？

是的！我们采用多重安全措施：
- 🔒 银行级 AES-256 加密
- 🛡️ 数据传输使用 HTTPS
- 🔐 严格的访问控制
- 📝 符合 PDPA（新加坡个人数据保护法）

### 会分享我的数据吗？

**绝不会！** 我们：
- ❌ 不会出售您的数据
- ❌ 不会与第三方分享（除非法律要求）
- ❌ 不会用于广告
- ✅ 只用于提供服务

详见[隐私政策](PRIVACY_POLICY.md)

### 照片会被保存吗？

**照片处理流程**:
1. 上传到安全服务器
2. AI 分析（5-10 秒）
3. 生成营养数据
4. **照片自动删除**（24 小时内）

我们只保存营养数据，不长期保存照片。

### 可以删除我的数据吗？

可以！您有权删除所有数据。

```
/delete account   # 删除账号和所有数据
/delete history   # 只删除历史记录
```

**注意**: 删除后无法恢复！

### 如何保护账号安全？

**安全建议**:
- 🔒 不要分享您的 WhatsApp 账号
- 📱 启用 WhatsApp 两步验证
- 🚫 不要点击可疑链接
- 👀 定期检查账号活动

### 检测到异常登录会怎样？

如果检测到异常登录（新设备、新位置等），我们会：
1. 🚨 立即通过 WhatsApp 通知您
2. 🔒 暂时锁定账号
3. ✅ 需要您确认是否为本人操作

## 技术问题

### 为什么识别失败？

**常见原因**:
- 📸 照片质量差（模糊、光线不足）
- 🍽️ 食物不清晰或被遮挡
- 🌐 网络连接问题
- ⚙️ 服务器繁忙

**解决方法**:
1. 重新拍摄清晰的照片
2. 检查网络连接
3. 稍后重试
4. 联系支持: `/support`

### 为什么没有收到回复？

**检查清单**:
- [ ] WhatsApp 是否正常运行
- [ ] 是否有网络连接
- [ ] 是否被 Vita AI 拉黑（不太可能）
- [ ] 是否在维护时间

**解决方法**:
1. 等待 1-2 分钟
2. 重新发送消息
3. 重启 WhatsApp
4. 联系支持

### 为什么每日总结没有发送？

**可能原因**:
- 📊 今天没有记录任何餐食
- 🔕 关闭了每日总结功能
- 🌐 网络问题
- ⚙️ 系统维护

**解决方法**:
```
/digest           # 手动请求今日总结
/settings         # 检查设置
```

### 为什么无法上传照片？

**可能原因**:
- 📁 照片太大（> 5 MB）
- 📱 WhatsApp 权限问题
- 🌐 网络不稳定

**解决方法**:
1. 压缩照片
2. 检查 WhatsApp 权限
3. 切换到 Wi-Fi
4. 重启 WhatsApp

### 为什么命令不起作用？

**检查**:
- ✅ 命令拼写正确
- ✅ 使用 `/` 开头
- ✅ 命令存在（发送 `/help` 查看）

**示例**:
```
✅ /history
❌ history
❌ /histroy
```

### 系统维护时间？

通常在凌晨 2:00-4:00 (SGT) 进行维护，持续 30 分钟左右。

维护期间：
- ❌ 无法识别食物
- ✅ 可以查看历史记录
- ✅ 可以使用基础命令

/bug
```

请提供：
- 📝 问题描述
- 📸 截图（如果有）
- 🕐 发生时间
