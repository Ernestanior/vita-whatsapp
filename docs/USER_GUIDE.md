## 快速开始

### 第一步：添加 Vita AI

1. 保存 Vita AI 的 WhatsApp 号码到您的联系人
2. 在 WhatsApp 中打开与 Vita AI 的对话
3. 发送 "Hi" 或 "/start" 开始

### 第二步：设置健康画像

Vita AI 会引导您完成健康画像设置：

```
1. 年龄: 例如 "30"
2. 性别: "男" 或 "女"
3. 身高: 例如 "175cm"
4. 体重: 例如 "70kg"
5. 活动水平: 
   - 久坐 (很少运动)
   - 轻度活动 (每周 1-3 天运动)
   - 中度活动 (每周 3-5 天运动)
   - 高度活动 (每周 6-7 天运动)
   - 非常活跃 (每天高强度运动)
6. 健康目标:
   - 减重
   - 增重
   - 维持体重
```

### 第三步：开始使用

设置完成后，您可以：

- 📸 发送食物照片进行营养分析
- 💬 使用命令查看历史记录和统计
- 📊 每天晚上 9 点接收健康总结

## 核心功能

### 1. 食物识别和分析

**如何使用**:
1. 拍摄食物照片
2. 发送给 Vita AI
3. 等待 5-10 秒
4. 收到详细的营养分析

**分析内容**:
- 🍽️ 食物名称和份量
- 🔢 营养成分（卡路里、蛋白质、碳水、脂肪、钠）
- 🚦 健康评级（绿灯/黄灯/红灯）
- 💡 个性化建议
- 📈 与每日目标的对比

**示例**:
```
🍚 Chicken Rice (1 plate)

营养成分:
• 卡路里: 500 kcal
• 蛋白质: 25g
• 碳水化合物: 60g
• 脂肪: 15g
• 钠: 800mg

健康评级: 🟢 绿灯
这是一份均衡的餐食！

建议:
• 已达到每日卡路里的 25%
• 蛋白质摄入良好
• 注意钠含量偏高

是否保存此记录？
[保存] [修改] [忽略]
```

### 2. 健康评级系统

**🟢 绿灯 (健康)**
- 营养均衡
- 符合您的健康目标
- 可以放心享用

**🟡 黄灯 (适度)**
- 某些营养素偏高或偏低
- 偶尔食用可以
- 注意份量控制

**🔴 红灯 (谨慎)**
- 高热量、高脂肪或高钠
- 不符合健康目标
- 建议选择更健康的替代品

### 3. 历史记录

**查看历史**:
```
发送: /history
```

**筛选记录**:
```
发送: /history today        # 今天的记录
发送: /history week         # 本周的记录
发送: /history month        # 本月的记录
```

**搜索食物**:
```
发送: /search chicken       # 搜索包含 "chicken" 的记录
```

### 4. 统计数据

**查看统计**:
```
发送: /stats
```

**统计内容**:
- 📊 本周营养摄入趋势
- 🎯 目标达成情况
- 🏆 连续打卡天数
- 📈 体重变化（如果有记录）

### 5. 每日健康总结

**自动发送时间**: 每天晚上 9:00 (SGT)

**总结内容**:
- 📊 今日营养摄入汇总
- 🎯 目标达成情况
- 💡 健康洞察和建议
- 🏃 推荐的运动量

**示例**:
```
📊 您的每日健康总结

今日统计:
• 记录餐食: 3 次
• 总卡路里: 1,800 kcal
• 健康评分: 85/100

洞察:
✅ 卡路里摄入适中
✅ 蛋白质摄入充足
⚠️ 钠摄入偏高 (建议 < 2000mg)

建议:
• 晚餐选择低钠食物
• 多喝水帮助排钠
• 增加蔬菜摄入

运动建议:
🏃 快走 30 分钟可消耗 150 kcal

继续保持！💪
```

### 6. 更新健康画像

**更新信息**:
```
发送: /profile
```

然后按照提示更新：
- 体重
- 活动水平
- 健康目标

**自然语言更新**:
```

发送: "我最近运动比较多"
```

## 常见问题

### 关于识别

**Q: 识别准确吗？**
A: Vita AI 使用先进的 AI 技术，准确率约 85-90%。如果识别不准确，您可以点击"修改"按钮进行调整。

**Q: 可以识别哪些食物？**
A: 支持新加坡常见食物，包括：
- 本地美食（鸡饭、叻沙、肉骨茶等）
- 中餐、西餐、日料、韩料
- 快餐、甜点、饮料
- 水果、零食

**Q: 识别需要多长时间？**
A: 通常 5-10 秒。高峰时段可能需要更长时间。

**Q: 可以识别多个食物吗？**
A: 可以！一张照片可以识别多个食物项目。

### 关于数据

**Q: 我的数据安全吗？**
A: 是的！我们使用银行级加密保护您的数据。详见[隐私政策](PRIVACY_POLICY.md)。

**Q: 可以导出数据吗？**
A: Premium 和 Pro 用户可以导出 CSV 格式，Pro 用户还可以导出 PDF 报告。

**Q: 可以删除历史记录吗？**
A: 可以。发送 `/delete` 并选择要删除的记录。

### 关于订阅

**Q: 可以随时取消吗？**
A: 可以！随时取消，无需理由。订阅将在当前周期结束后停止。

**Q: 取消后还能使用吗？**
A: 可以继续使用 Free Plan 的功能。

**Q: 支持哪些支付方式？**
A: 支持信用卡、借记卡、PayNow。

## 技巧和窍门

### 1. 拍摄技巧

**最佳拍摄方式**:
- 📸 光线充足
- 📐 正面拍摄
- 🔍 食物清晰可见
- 📏 包含参照物（如餐具）

**避免**:
- ❌ 光线太暗
- ❌ 角度太斜
- ❌ 食物被遮挡
- ❌ 照片模糊

### 2. 提高准确性

**提供更多信息**:
```
发送照片后，可以补充说明：
"这是一份大份的鸡饭"
"这是半份"
"这是儿童份"
```

**修改识别结果**:
```
点击 [修改] 按钮
调整食物名称或份量
保存修改
```

### 3. 充分利用建议

**个性化建议**:
- 根据您的健康目标定制
- 考虑您的活动水平
- 基于您的饮食历史

**如何应用**:
- 📝 记录建议
- 🎯 设定小目标
- 📊 追踪进度

### 4. 设定目标

**SMART 目标**:
- Specific (具体): "每天摄入 1800 卡路里"
- Measurable (可衡量): 使用 Vita AI 追踪
- Achievable (可实现): 根据您的情况设定
- Relevant (相关): 符合健康目标
- Time-bound (有时限): "30 天内减重 2kg"

### 5. 保持动力

**连续打卡**:
- 🔥 每天记录至少一餐
- 🏆 解锁成就徽章
- 📈 查看进步曲线

**社交支持**:
- 👥 邀请朋友一起使用
- 💬 分享您的成就
- 🎉 庆祝里程碑
